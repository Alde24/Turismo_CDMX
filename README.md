# Itinerarios Turísticos CDMX - Sistema de Recomendación

## Descripción del Proyecto
Plataforma web diseñada para generar itinerarios de viaje personalizados dentro de la Ciudad de México. El sistema analiza las preferencias y gustos del usuario para ofrecer una ruta turística optimizada. Este proyecto fue desarrollado como entrega final para la materia de **Análisis y Diseño de Sistemas** en la Escuela Superior de Cómputo (ESCOM-IPN).

## Tecnologías Utilizadas
* **Frontend:** HTML5, CSS3, JS
* **Backend:** PHP
* **Base de Datos:** MySQL/SQL

## Características Principales
* **Personalización:** Generación de rutas basadas en los gustos y preferencias del usuario.
* **Interfaz intuitiva:** Diseño enfocado en la experiencia de usuario para la navegación turística.
* **Gestión de Datos:** Arquitectura diseñada para el manejo eficiente de puntos de interés turístico.
